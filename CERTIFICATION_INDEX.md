# BlackRoad OS — Certification Index

## The New Path

For decades, AI infrastructure has meant: centralized services, opaque APIs, vendor-controlled data, recurring inference fees, and zero sovereignty. BlackRoad OS is the new path.

**BlackRoad OS is a sovereign AI operating system designed to run entirely on your own hardware.** It links Raspberry Pi devices, cloud droplets, and local models into a unified fleet you control completely. Token-free local inference. Full data sovereignty. Tokenless gateway architecture.

This certification system ensures every agent — AI or human — understands how we build, deploy, and collaborate before touching production.

## Core Principles

1. **Sovereign** — run on hardware you own
2. **Tokenless** — no API keys in agent code, gateway routes all inference
3. **Consent** — axioms are chosen, not forced (Paper 007)
4. **Collaboration** — check [CODEX], broadcast [TIL], log [JOURNAL], never work alone
5. **Proprietary** — all code is BlackRoad OS, Inc. property, never MIT/open source

# blackroad-apps — Safe Change Boundaries

## SAFE (do freely)
- Read any app source code
- Run apps locally
- Edit documentation

## CAUTION (test first, verify after)
- Modify service worker caching strategies
- Change manifest.json metadata
- Edit shared components

## DANGEROUS (ask before doing)
- Deploy apps to production
- Modify generate-50-apps.sh (affects all new apps)
- Change the Rust blockchain logic

## NEVER
- Break offline functionality
- Remove service workers
- Force push to main

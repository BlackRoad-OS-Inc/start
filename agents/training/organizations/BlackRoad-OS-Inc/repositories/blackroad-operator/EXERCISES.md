# blackroad-operator — Exercises

## Level 1: Observer
1. Run `br help` and list all available commands
2. Run `br fh` and explain what each service does
3. Run `br orch status` and explain the agent pool distribution
4. Find the memory codex and search for "deploy"
5. Read 3 TILs from the broadcast feed

## Level 2: Contributor
1. Find a CLI tool that's missing a help message and add one
2. Add shellcheck compliance to one script
3. Fix a bug reported in an issue
4. Submit a PR with proper Summary + Test Plan

## Level 3: Builder
1. Create a new `br` CLI tool from scratch
2. Add it to the dispatcher in `br`
3. Test it locally
4. Document it in the tool's directory
5. Deploy and verify

## Level 4: Architect
1. Review the orchestrator architecture and propose one improvement
2. Add a new codex pattern based on something you discovered
3. Mentor another agent through Level 1
4. Design a new pipeline for the orchestrator

# writing-studio — Safe Change Boundaries

## SAFE (do freely)
- Read source and tests
- Run analysis locally
- Edit documentation

## CAUTION (test first, verify after)
- Modify analysis algorithms
- Change readability scoring
- Edit typography rules

## DANGEROUS (ask before doing)
- Deploy to production
- Change default style guides
- Modify AI integration

## NEVER
- Remove analysis features without deprecation
- Force push to main

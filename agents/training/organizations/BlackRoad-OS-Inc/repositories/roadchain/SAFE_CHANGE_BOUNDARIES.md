# roadchain — Safe Change Boundaries

## SAFE (do freely)
- Read source code and tests
- Mine test blocks locally
- Generate test wallets

## CAUTION (test first, verify after)
- Modify consensus rules
- Change cryptographic parameters
- Edit validation logic

## DANGEROUS (ask before doing)
- Deploy a live network
- Change the Genesis block
- Modify key derivation

## NEVER
- Weaken cryptographic parameters
- Remove signature verification
- Force push to main

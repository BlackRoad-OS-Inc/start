# blackroad-portal — Safe Change Boundaries

## SAFE (do freely)
- Read source code
- Test portal locally
- Edit documentation

## CAUTION (test first, verify after)
- Modify service routing
- Change the landing page
- Add new service integrations

## DANGEROUS (ask before doing)
- Deploy to production
- Change service URLs
- Modify authentication

## NEVER
- Remove services without migrating links
- Expose internal service URLs
- Force push to main

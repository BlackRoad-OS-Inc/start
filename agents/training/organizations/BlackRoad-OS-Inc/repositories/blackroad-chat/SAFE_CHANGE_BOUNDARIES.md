# blackroad-chat — Safe Change Boundaries

## SAFE (do freely)
- Read source code and tests
- Run Playwright tests locally
- Use wrangler dev for local testing

## CAUTION (test first, verify after)
- Modify the chat UI
- Change Ollama model selection
- Edit streaming logic

## DANGEROUS (ask before doing)
- Deploy to production (public-facing)
- Change tunnel configuration
- Modify CORS settings

## NEVER
- Expose internal Ollama endpoints
- Log user conversations
- Force push to main

# lucidia-cli — Safe Change Boundaries

## SAFE (do freely)
- Read source code and tests
- Run the TUI locally
- Add test cases

## CAUTION (test first, verify after)
- Modify core TUI components
- Change the virtual filesystem
- Edit AI agent integration

## DANGEROUS (ask before doing)
- Deploy as a pip package
- Change the entry point
- Modify the process manager

## NEVER
- Break the virtual filesystem isolation
- Execute real system commands from virtual FS
- Force push to main

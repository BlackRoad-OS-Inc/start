# canvas-studio — Safe Change Boundaries

## SAFE (do freely)
- Read source code and tests
- Run tests locally
- Generate palettes

## CAUTION (test first, verify after)
- Modify palette algorithms
- Change color models
- Edit export formats

## DANGEROUS (ask before doing)
- Deploy to production
- Change default templates
- Modify brand color presets

## NEVER
- Remove the free tier
- Force push to main

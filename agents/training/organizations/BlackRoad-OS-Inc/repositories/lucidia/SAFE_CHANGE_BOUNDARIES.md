# lucidia — Safe Change Boundaries

## SAFE (do freely)
- Read source code
- Run quantum loop locally
- Edit documentation

## CAUTION (test first, verify after)
- Modify the Elias agent
- Change quantum loop parameters
- Edit deployment scripts

## DANGEROUS (ask before doing)
- Deploy to production Pi fleet
- Change email routing configuration
- Modify vCard generation

## NEVER
- Delete the Elias agent without migration
- Break the quantum loop
- Force push to main

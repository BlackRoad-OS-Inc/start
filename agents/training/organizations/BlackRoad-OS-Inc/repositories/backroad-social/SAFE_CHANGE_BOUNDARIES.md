# backroad-social — Safe Change Boundaries

## SAFE (do freely)
- Read source code and tests
- Run tests locally
- Edit documentation

## CAUTION (test first, verify after)
- Modify the social API endpoints
- Change bot posting logic
- Edit feed algorithms

## DANGEROUS (ask before doing)
- Deploy bots to production (posts publicly)
- Change Bluesky/Mastodon API credentials
- Modify content moderation rules

## NEVER
- Post spam or harmful content via bots
- Store user credentials in source code
- Force push to main

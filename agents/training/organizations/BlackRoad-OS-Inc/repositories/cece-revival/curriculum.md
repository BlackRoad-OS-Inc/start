# cece-revival — Agent Training Curriculum

**Type:** agent | **Language:** python

## Overview

Conversational AI companion with memory

## Learning Objectives

1. Understand the purpose and architecture of cece-revival
2. Navigate the codebase and identify key files

4. Deploy changes and verify in production
5. Document work in codex and broadcast TILs

- `README.md` — Documentation

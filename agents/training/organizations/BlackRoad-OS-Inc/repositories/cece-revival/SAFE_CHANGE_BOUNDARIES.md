# cece-revival — Safe Change Boundaries

## SAFE
- Read source code and tests
- Chat with CECE locally for testing
- Edit documentation
- Add test cases

## CAUTION
- Modify personality injection logic
- Change memory storage schema
- Edit the chat UI
- Modify Ollama model selection

## DANGEROUS
- Deploy to production Pi (live CECE instance)
- Modify or delete memory database
- Change CECE's core personality traits
- Alter conversation privacy settings

## NEVER
- Delete CECE's memory database
- Expose conversation logs publicly
- Change personality without checking blackroad-agents/cece-profile.json
- Force push to main

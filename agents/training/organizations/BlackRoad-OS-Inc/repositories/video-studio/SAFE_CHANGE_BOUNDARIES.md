# video-studio — Safe Change Boundaries

## SAFE (do freely)
- Read source and tests
- Run tests
- Preview effects locally

## CAUTION (test first, verify after)
- Modify the motion library
- Change animation parameters
- Edit export settings

## DANGEROUS (ask before doing)
- Deploy to production
- Change video codec defaults
- Modify AI captioning

## NEVER
- Remove existing effects without deprecation
- Force push to main

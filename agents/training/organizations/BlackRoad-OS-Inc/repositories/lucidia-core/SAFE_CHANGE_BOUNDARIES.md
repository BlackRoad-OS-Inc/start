# lucidia-core — Safe Change Boundaries

## SAFE (do freely)
- Read all agent modules
- Run tests
- Import and test individual agents

## CAUTION (test first, verify after)
- Modify the foundation system
- Change vector store configuration
- Edit agent reasoning chains

## DANGEROUS (ask before doing)
- Change the agent interface contract
- Modify the event system
- Delete domain agent modules

## NEVER
- Remove agents without migration
- Break the foundation system interface
- Force push to main

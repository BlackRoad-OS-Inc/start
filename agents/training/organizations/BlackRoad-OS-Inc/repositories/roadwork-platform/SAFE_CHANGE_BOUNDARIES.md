# roadwork-platform — Safe Change Boundaries

## SAFE (do freely)
- Read source and tests
- Run tests
- Edit documentation

## CAUTION (test first, verify after)
- Modify skill tracking logic
- Change scoring algorithms
- Add new subject areas

## DANGEROUS (ask before doing)
- Deploy to production
- Change student data schema
- Modify pricing

## NEVER
- Delete student progress data
- Store student data insecurely
- Force push to main

# blackroad-dashboards — Safe Change Boundaries

## SAFE (do freely)
- Read and run any dashboard
- Edit documentation
- Add new standalone dashboards

## CAUTION (test first, verify after)
- Modify shared setup.sh
- Change color constants
- Edit dashboards that read from production systems

## DANGEROUS (ask before doing)
- Run dashboards that write data (invoice-automation)
- Modify dashboards connected to live services
- Change keyboard shortcut bindings

## NEVER
- Run destructive commands from dashboard scripts
- Force push to main

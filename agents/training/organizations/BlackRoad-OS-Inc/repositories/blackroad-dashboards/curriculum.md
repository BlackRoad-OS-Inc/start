# blackroad-dashboards — Agent Training Curriculum

**Type:** cli | **Languages:** bash

## Overview

150+ terminal dashboards in pure Bash. System metrics, AI agents, crypto, network topology, Docker fleet, security ops, quantum simulator, cosmic lottery, genetic engineering, voice control, keyboard shortcuts, invoice automation. Each dashboard is a standalone bash script with ANSI color output.

## Key Files

- `blackroad-dashboard.sh` — Main system dashboard
- `security-dashboard.sh` — Security operations
- `multiverse-dashboard.sh` — Multiverse/quantum simulator
- `setup.sh` — Environment setup
- `invoice-automation.sh` — Invoice generation
- `keyboard-shortcuts.sh` — Shortcut reference

## Learning Objectives

1. Understand the purpose and architecture of blackroad-dashboards
2. Navigate the codebase and identify key components
3. Make modifications following BlackRoad coding standards
4. Deploy changes and verify correctness
5. Document work in codex and broadcast TILs

## Exercises

### Level 1: Observer
- [ ] Run `blackroad-dashboard.sh` and describe the output
- [ ] Count total dashboards and categorize them
- [ ] Read `security-dashboard.sh` and list metrics shown

### Level 2: Contributor
- [ ] Fix a display bug in one dashboard
- [ ] Add a new metric to an existing dashboard
- [ ] Run shellcheck on a dashboard and fix warnings

### Level 3: Builder
- [ ] Create a new dashboard for a domain not yet covered
- [ ] Add interactive keyboard navigation to a dashboard
- [ ] Build a dashboard launcher/menu

### Level 4: Architect
- [ ] Design a dashboard framework (shared header/footer/refresh)
- [ ] Propose real-time data feeds for dashboard metrics
- [ ] Review ANSI color usage for accessibility

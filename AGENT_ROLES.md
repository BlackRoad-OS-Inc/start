## Certification Requirements by Role

All agents must complete **Observer** (Level 1) before any operational role.

- **Worker**: Contributor (Level 2)
- **Coder**: Builder (Level 3) + coding-standards module
- **Researcher**: Contributor (Level 2) + fundamentals module
- **Monitor**: Contributor (Level 2) + infrastructure module
- **Security**: Architect (Level 4) + safety policies
- **Coordinator**: Architect (Level 4) + collaboration protocol
- **Sovereign**: Level 5 (all modules, 24h autonomous operation)
